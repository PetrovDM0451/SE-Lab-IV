Who We Are

This website was created as part of the Software Engineering Lab course. Our goal is to practice collaborative development using Git and GitHub. Each team member is responsible for building one section of the website.

Our Mission

We aim to learn how software teams collaborate in real-world projects. By using version control systems like Git and GitHub, we can work on different features simultaneously and combine them into a single project.
